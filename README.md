This project is a detailed **UI (User Interface) clone** of the Trip Advisor homepage. It was developed to practice and master modern front-end layout techniques, primarily **CSS Flexbox** and **CSS Grid**, while focusing on building a highly **responsive** and reusable component-based structure.
